# status_social_media_addiction
Projeto de análise de dados que demonstra o status da análise de vício em mídias sociais usando Python, SQL e Power BI.

Este projeto consiste em analisar o status de vício em mídias sociais dos alunos nas redes sociais e os resultados relacionados
em suas vidas. Abrange vários países e níveis acadêmicos, com foco em dimensões-chave como intensidade de uso, preferências de plataforma e dinâmica de relacionamentos.

# Ferramentas
Para a execução dessa análise, foram utilizadas algumas ferramentas, tais quais:

* SQL: As consultas foram estruturadas através do DBeaver, utilizando o PostgreSQL.
* Python: Para realizar editar informações da tabela e fazer a conexão das tabelas .csv e o PostgreSQL.
* Microsoft Power BI: O dashboard do projeto foi construído com o uso do software, com a necessidade da criação de algumas funções em DAX para a construção das visualizações.
